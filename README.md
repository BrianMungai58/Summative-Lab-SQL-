Purpose
This lab demonstrates SQL querying, data analysis with Pandas, and data visualization on two datasets:
1. A wholesale product sales database (customer and order data).
2. An IMDB movie database (movies released between 2010–2019).

Part 1 Highlights
1.Filtered customers by region and credit limits.
2.Identified top customers by total payments.
3.Analyzed product line performance (quantity vs. diversity).
4.Created bar charts and scatter plots to support findings.

Part 2 Highlights
1.Explored genre distribution and average ratings.
2.Formulated the business question: *"Do genres with more releases have higher average ratings?"*
3.Identified key data cleaning tasks (missing values, splitting genres, vote thresholds).

Files Included
1.SQLSummativeLab.ipynb` – The main notebook with all code and outputs.
2.README.md` – This file.

Run all cells in the notebook. The required SQLite databases (`data.sqlite` and `im.db`) will be loaded or extracted automatically.
